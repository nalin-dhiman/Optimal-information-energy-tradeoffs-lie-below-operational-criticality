
## Directory structure

- `manuscript/`
  - `figures/` : figures used in the manuscript and SI
  - `tables/` : CSV tables copied from revision analyses
- `code_submission/`
  - `renormalized_criticality_clean/` : cleaned code and results snapshot
    - includes `revision_analysis_v2/` and `revision_analysis_v3/`

## Main scientific claim

The manuscript argues, conservatively, that optimal information--energy tradeoffs occupy a finite **subcritical regime** below operational criticality in the studied driven renewal network. It does **not** claim a universal constant offset or a clean asymptotic scaling law.

## Reproducibility

The packaged code snapshot contains the simulation/optimization pipeline and the reviewer-driven analyses. The `reproduce_figures.sh` helper notes the key source directories used for the manuscript figures.
